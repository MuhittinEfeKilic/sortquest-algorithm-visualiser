Deployment Instructions
Requirements
Any modern web browser:
Google Chrome
Mozilla Firefox
Microsoft Edge
Safari

No additional software, package manager, database, or server is required.

Local Execution
Download or clone the repository.
git clone https://github.com/MuhittinEfeKilic/sortquest-algorithm-visualiser
Navigate to the project folder.
Open index.html in a modern web browser.

The game will start immediately.

Verification

After opening index.html, verify that:

The SortQuest main menu is displayed.
Game modes can be selected.
Tiles can be interacted with.
Scoring and gameplay mechanics function correctly.
Alternative Deployment

The project can also be hosted on any static web hosting service, including:

GitHub Pages
Netlify
Vercel
Firebase Hosting

Simply upload all source files while preserving the directory structure.

No backend services are required.
