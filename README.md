# SortQuest: Tile Puzzle Algorithm Visualiser

A browser-based educational game developed for SEN4012 Analysis of Algorithms.

## Team
- Muhittin Efe Kılıç
- Eray Çalışkan

## How to Run

1. Download the repository.
2. Open `index.html` in any modern browser.
3. No installation or server setup is required.

## Technologies
- HTML5
- CSS3
- JavaScript
